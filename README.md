# 💧 Spending Tracker

Ứng dụng quản lý chi tiêu cá nhân được xây dựng bằng React Native + Expo.

## ✨ Tính năng

- 📊 Theo dõi thu/chi theo danh mục
- 🤖 AI tự động phân loại giao dịch (Gemini API)
- 💳 Quản lý khoản nợ với progress bar
- 🎯 Đặt ngân sách và cảnh báo vượt mức
- 📈 Thống kê chi tiêu theo tháng
- 🌙 Dark mode — thiết kế theo màu mệnh Thủy

## 🛠 Tech Stack

- **Framework:** React Native + Expo SDK 54
- **Navigation:** Expo Router
- **Database:** SQLite (expo-sqlite)
- **State Management:** Zustand
- **AI:** Google Gemini API
